## Instructions to run

* Pre-requisites:
	-  Py3
	-  Tor

* < directions to install > 
```bash
pip install -r requirements.txt
```

* < directions to execute >

```bash
# start tor on port 9150
pproxy -l http://:8181 -r socks5://127.0.0.1:9150 -vv
scrapy crawl name_of_spider # DRL
```
